# Torrents to IDM

This project allows you to download torrent files and transfer them to Internet Download Manager (IDM) for faster downloads.

## Features
- Download torrent files
- Integrate with IDM for high-speed downloads

## Requirements
- Node.js (version 14.0 or higher)
- Internet Download Manager (IDM)

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/russellpeiris/torrents-to-idm.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```

## Usage
1. Start the server:
   ```bash
   node server.js
   ```
2. Follow the instructions in the terminal or web interface to add torrent files and transfer them to IDM.

## Configuration
- Edit `server.js` to customize settings as needed.
